### Installation Instructions
1. Clone this repository
2. Install latest version of Android Studio
3. Configure any PATH variables (depending on OS)
4. Open > File and browse to the cloned version of this repository
5. Allow gradle sync to run
6. From Android Studio:
`Build → Make Project`
`Run → select device/emulator → Run 'app'`
- or -
From terminal:
macOS/Linux: `./gradlew assembleDebug`
Windows: `gradlew.bat assembleDebug`
